使用 `sudo n stable` 命令更新 node 时突然报错提示 `Killed 9`，同样，使用 npm 命令时也一样，网上找不到有效的解决方案，经过一番尝试终于解决。

<!-- more -->

解决办法其实很简单，到 Nodejs 官网下载最新的 Nodejs 安装包重新安装即可。
